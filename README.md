# eqvskursia
# 🚌 ექსკურსიის ავტობუსების გაქირავების სისტემა (Bus Rental Platform)

Full-Stack (MERN) ვებ-აპლიკაცია, რომელიც განკუთვნილია ექსკურსიებისთვის, ტურებისთვის და ჯგუფური მგზავრობისთვის ავტობუსებისა და მიკროავტობუსების დასაჯავშნად.

პლატფორმა მომხმარებელს საშუალებას აძლევს შეარჩიოს სასურველი ტრანსპორტი ტევადობის მიხედვით, გაიგოს ფასი მანძილის/დღეების მიხედვით და გაგზავნოს ჯავშნის მოთხოვნა.

---

## 🛠️ ტექნოლოგიური სტეკი (Tech Stack)

* **Frontend:** React.js, TypeScript, Tailwind CSS, Vite
* **Backend:** Node.js, Express.js
* **მონაცემთა ბაზა:** MongoDB (Mongoose)
* **ინსტრუმენტები:** Bun / npm, Git

---

## ✨ ფუნქციონალი (Features)

* **ავტოპარკის კატალოგი:** ავტობუსების და მიკროავტობუსების სიის ჩვენება (სურათები, სავარძლების რაოდენობა, კონდიცირება, WiFi და ა.შ.).
* **კალკულატორი & ჯავშნის ფორმა:** მგზავრების რაოდენობის, გასვლის/დაბრუნების თარიღებისა და მარშრუტის მითითება (`BookingForm`).
* **ბექენდ API:** მომხმარებლის მიერ გამოგზავნილი ჯავშნების მიღება, ვალიდაცია და შენახვა ბაზაში.
* **სტატუსის მართვა:** შემოსული მოთხოვნების დამუშავება ადმინისტრირებისთვის.
